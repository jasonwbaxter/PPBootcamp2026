# About This Site

Welcome! This site is built as part of **PPBootcamp 2026** — a hands-on programme focused on designing high-impact prompts and workflows with Microsoft 365 Copilot and GitHub Copilot.

## Our Mission

We believe AI-assisted tools should feel natural and empowering for every developer and knowledge worker. This site serves as a living example of how Markdown-powered content management can be set up quickly and maintained sustainably.

## What You'll Find Here

- **Posts** – Articles covering Copilot tips, Markdown techniques, and workflow automation
- **Pages** – Reference material, guides, and project documentation
- **Templates** – Reusable starting points for new content

## The Tech Stack

| Layer | Choice |
|-------|--------|
| Content format | Markdown with YAML front matter |
| Version control | Git / GitHub |
| AI assistance | GitHub Copilot |
| Build tooling | Scripts in `/scripts` |
